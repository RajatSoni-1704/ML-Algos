# Machine Learning Algorithms from Scratch

This repository contains my personal implementations of foundational machine learning algorithms from scratch. The primary goal of this project is to deepen my understanding of how these algorithms work under the hood, mathematically and programmatically, without relying on high-level libraries like scikit-learn.

## 📁 Repository Structure

- **`Regression/`**: Contains regression models (e.g., Linear Regression).
- **`Dimensionality_Reduction/`**: Contains techniques like Principal Component Analysis (PCA).
- **`Micrograd/`**: An exploration of automatic differentiation and neural networks (inspired by Andrej Karpathy's micrograd).
- **`Classification/`**: (Coming soon) Will contain classification algorithms like Logistic Regression, KNN, etc.
- **`Datasets/`**: Datasets used for testing the implementations.

## 🚀 Purpose

- **Learning**: To gain a concrete understanding of ML algorithms by building them from the ground up.
- **Practice**: To improve Python programming skills and familiarity with numerical libraries like NumPy.
- **Reference**: To serve as a personal reference for how core algorithms are implemented.

## 🛠️ Technologies Used

- Python
- NumPy (for linear algebra and numerical operations)
- Matplotlib/Seaborn (for data visualization)
- Jupyter Notebooks (forexperimentation experimentation and demonstrations)

## 💡 How to Use

Feel free to explore the code! Most folders contain Python files (`.py`) with the core algorithm classes/functions, and Jupyter notebooks (`.ipynb`) demonstrating their usage on toy datasets.
